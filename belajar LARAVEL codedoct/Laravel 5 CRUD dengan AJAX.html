Laravel 5 CRUD dengan AJAX

admin 02 Dec 2016 07:38 am Website Framework Jquery Laravel mysql PHP

Laravel 5 CRUD dengan AJAX sehingga tidak perlu merefreh halaman secara untuh yang direfresh hanya pada bagian tertentu yang berhubungan dengan Ajax, Laravel 5 sangat fleksibel seperti halnya dapat dikombinasikan dengan Ajax.

Sebelumnya saya sudah pernah membuat sebuah artikel tentang Laravel 5 dan Angular Js CRUD Lengkap dengan Pencarian dan Pagination dimana hal yang mirip dengan Ajax karena sama-sama menggunakan javascript tetapi pada kesempatan kali ini kita dapat melihat mana yang lebih mudah untuk diimplementasikan dalam pemrograman Laravel.

Daripada penasaran mari kita langsung coba mempraktekannya dengan mengikuti langkah-langkah berikut ini :

Persiapan Awal

1. Asumsi saya anda sudah mengunduh Laravel 5.2 baca cara Mengunduh Laravel

2. Mengunduh bootstrap.min.css, jquery.min.js, bootstrap.min.js silahkan unduh di Laravel Ajax

3. Ekstrak file yang sudah di unduh kedalam folder \public.

Langkah 1 Membuat Database dan Setting Koneksi

Buatlah database dengan nama laravel pada server, pada contoh ini saya buat pada XAMPP Server

Lakukan setting koneksi pada file .ENV yang ada pada root project laravel yang sudah diunduh atau yang sudah dimiliki dan sesuaikan dengan settingan server anda seperti contoh berikut

DB_CONNECTION=mysql
DB_HOST=127.0.0.1
DB_PORT=3306
DB_DATABASE=laravel
DB_USERNAME=default XAMPP adalah (root)
DB_PASSWORD=default XAMPP adalah (null)

Langkah 2 Membuat Migrasi Tabel

Buka Command Prompt atau ketik CMD (bagi pengguna Windows) dan masuk ke direktori project laravel misalnya untuk membuka D:\LARAVEL\ArsipAngular>
maka penrintahnya adalah sebagai berikut :

Microsoft Windows [Version 10.0.10240]
(c) 2015 Microsoft Corporation. All rights reserved.

C:\Users\USER>d:
D:\>cd laravel\arsipAjax
D:\LARAVEL\ArsipAjax>

Lakukan perintah berikut untuk membuat skema tabel tags

php artisan make:migration BuatTabelTag --create=tags
//hasilnya pada folder database\migrations
Created Migration: 2016_11_22_021245_BuatTabelTag

maka akan terbuat sebuah file baru pada folder database\migrations, buka file tersebut tambahkan 2 field lalu simpan, perhatikan code berikut

    public function up()
    {
        Schema::create('tags', function (Blueprint $table) {
            $table->increments('id');
            //tambahkan 2 field
            $table->string('tagname');
            $table->text('slug');
            //
            $table->timestamps();
        });
    }

setelah menambahkan field pada skema tabel tags lakukan perintah berikut untuk melakukan migrasi tabel data agar tabel terbuat pada database

php artisan migrate
//hasil migrasi tabel
Migrated: 2016_11_22_021245_BuatTabelTag

 sampai pada langkah ini database laravel sudah siap dengan tabel tags, silahkan buka dan periksa database anda

Langkah 3 Membuat Model

Membuat model setiap tabel diperlukan untuk menghubungkan tabel yang ada pada database dengan aplikasi

php artisan make:model Tag

maka akan terbuat sebuah file Tag.php pada folder \app, untuk itu lakukan perubahan sehingga nampak seperti berikut

<?php

Langkah 4 Membuat Controller

Pada aplikasi ini controller di butuhkan untuk mengontrol setiap perintah yang dikirim dari sisi client seperti CRUD, oleh karena itu buatlah sebuah controller dengan nama TagController, dengan code pada command prompt yang sama

php artisan make:controller TagController
//hasil
Controller created successfully.

maka akan terbuat sebuah file TagController.php pada folder \app\Http\Controllers, buka file tersebut lakukan perubahan seperti code di bawah ini

<?php
namespace App\Http\Controllers;
use Illuminate\Http\Request;
use App\Http\Requests;
use App\Tag; //<--Jangan sampai lupa nambahin use App\Tag;

class TagController extends Controller
{
    //
    public function index()
    {
        $tags = Tag::all();
        return view('ajax.ajax')->with('tags',$tags);
    }

    public function store(Request $request)
    {
        $tag = Tag::create($request->input());
        return response()->json($tag);
    }

    public function edit($tag_id)
    {
        $tag = Tag::find($tag_id);
        return response()->json($tag);
    }

    public function update(Request $request,$tag_id)
    {
        $tag = Tag::find($tag_id);
        $tag->tagname = $request->tagname;
        $tag->slug = $request->slug;
        $tag->save();
        return response()->json($tag);
    }

    public function destroy($tag_id)
    {
        $tag = Tag::destroy($tag_id);
        return response()->json($tag);

    }
}

Langkah 5 Memperbaharui route

routes.php menurut pemahaman saya kira-kira berfungsi untuk mententukan jalur dan mengarahkan dari setiap permintaan dari client, dengan demikian wajib kita menentukan jalur tersebut. silahkan buka file routes.php yang ada pada folder  \app\Http dan sesuaikan dengan kode berikut :

Route::get('/', function () {
    return view('ajax.beranda');
});
Route::get('LaravelAjaxCRUD/','TagController@index');
Route::post('LaravelAjaxCRUD/','TagController@store');
Route::get('LaravelAjaxCRUD/{tag_id?}','TagController@edit');
Route::put('LaravelAjaxCRUD/{tag_id?}','TagController@update');
Route::delete('LaravelAjaxCRUD/{tag_id?}','TagController@destroy');

Langkah 7 Membuat File AjaxScript

file ajaxscript.js ini sudah ada pada folder js yang sudah di unduh sebelumnya pada tahap persiapan, jika sudah diekstrak maka file ini berada pada folder public\js\ dengan sintak sebagai berikut

/**
 * Created by ArsipTI.com on 01/12/2016.
 */
var url = "http://localhost:8000/LaravelAjaxCRUD";

//Menampilkan Form  dan Data
$(document).on('click','.open_modal',function(){
    var tag_id = $(this).val();

    $.get(url + '/' + tag_id, function (data) {
        //success data
        console.log(data);
        $('#tag_id').val(data.id);
        $('#tagname').val(data.tagname);
        $('#slug').val(data.slug);
        $('#btn-save').val("update");
        $('#myModal').modal('show');
    })
});
//Menampilkan Isian Form
$('#btn_add').click(function(){
    $('#btn-save').val("add");
    $('#frmTags').trigger("reset");
    $('#myModal').modal('show');
});
//Menghapus data
$(document).on('click','.delete-tag',function(){
    var tag_id = $(this).val();
    $.ajaxSetup({
        headers: {
            'X-CSRF-TOKEN': $('meta[name="_token"]').attr('content')
        }
    })
    $.ajax({
        type: "DELETE",
        url: url + '/' + tag_id,
        success: function (data) {
            console.log(data);
            $("#tag" + tag_id).remove();
        },
        error: function (data) {
            console.log('Error:', data);
        }
    });
});
//membuat tag / update tag
$("#btn-save").click(function (e) {
    $.ajaxSetup({
        headers: {
            'X-CSRF-TOKEN': $('meta[name="_token"]').attr('content')
        }
    })
    e.preventDefault();
    var formData = {
        tagname: $('#tagname').val(),
        slug: $('#slug').val(),
    }
    //Jika Tombol Simpan ditekan/diklik [tambah=POST], [perbaharui=PUT]
    var state = $('#btn-save').val();
    var type = "POST"; //membuat data baru
    var tag_id = $('#tag_id').val();;
    var my_url = url;
    if (state == "update"){
        type = "PUT"; //memperbaharui data yang sudah ada
        my_url += '/' + tag_id;
    }
    console.log(formData);
    $.ajax({
        type: type,
        url: my_url,
        data: formData,
        dataType: 'json',
        success: function (data) {
            console.log(data);
            var tag = '<tr id="tag' + data.id + '"><td>' + data.id + '</td><td>' + data.tagname + '</td><td>' + data.slug + '</td>';
            tag += '<td><button class="btn btn-warning btn-detail open_modal" value="' + data.id + '">Edit</button>';
            tag += ' <button class="btn btn-danger btn-delete delete-tag" value="' + data.id + '">Delete</button></td></tr>';
            if (state == "add"){ //jika data baru ditambahkan
                $('#tags-list').append(tag);
            }else{ //jika data diperbaharui
                $("#tag" + tag_id).replaceWith( tag );
            }
            $('#frmTags').trigger("reset");
            $('#myModal').modal('hide')
        },
        error: function (data) {
            console.log('Error:', data);
        }
    });
});
